# insider-beta

Prototype of a political trading terminal.

## Why this exists

I want a tool that tracks political events and lets me trade or simulate 
trades
based on those signals, instead of only using price and fundamentals.

## Current status

- [x] Initial repo and Git setup
- [x] v0.1 — basic watchlist CRUD
- [ ] Connect to real data sources
- [ ] Portfolio and P&L view
- [ ] Backtesting for strategies

## Tech stack (planned)

- Backend: Python
- Frontend: Next.js
- Database: TBD

## How to run locally

```bash
npm install
npm run dev
```
